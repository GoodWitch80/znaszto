# ZnaszTo — Kompetencje cyfrowe dla nauczycieli

Polish platform for teachers on digital competencies (ICT/TIK, LLM, AI) with a free, no-login AI lesson generator. Static multi-page site (HTML/CSS/vanilla JS, no backend, no API keys).

## Co zawiera / What's inside

- **Generator lekcji AI** (`/generator`) — client-side generator producing konspekt (lesson plan), karta pracy (worksheet) and zadanie domowe (homework), with print/PDF export. Runs fully in the browser, no backend, no API key.
- 6 educational modules: Narzędzia TIK, LLM w szkole, AI w pracy, Scenariusze, Współpraca, O projekcie
- **Blog** with a full article + JSON-LD (`BlogPosting`, `FAQPage`, `BreadcrumbList`) + FAQ + "Powiązane pytania" block
- Privacy & Terms pages
- Accessibility: skip link, semantic landmarks, dark mode toggle, focus styles, `prefers-reduced-motion`
- Security headers in `_headers` (CSP, Permissions-Policy, HSTS, X-Frame-Options)
- `sitemap.xml`, `robots.txt`, OG image, favicon

## Build

The HTML pages are generated from `build.js`:

```bash
node build.js            # uses default BASE_URL
BASE_URL=https://your-domain.com node build.js   # set your domain
```

`BASE_URL` controls absolute canonical/og:url tags, sitemap and robots.txt. Update it to your final domain, rebuild, commit, and push — Netlify auto-redeploys.

## Deploy

### Netlify (recommended)
1. Connect this repo to Netlify.
2. Build command: *(none)*. Publish directory: `.` (repo root).
3. Site goes live at `<site-name>.netlify.app`. `netlify.toml` and `_headers` are already configured.

### GitHub Pages
Links use clean URLs (`/generator`). GitHub Pages does not rewrite these to `.html` by default, so prefer Netlify, or adjust links to include `.html`.

## Tech
Vanilla HTML/CSS/JS, generated by `build.js` (Node). No framework, no build step for serving.
